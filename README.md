# Curso de VSCode

Un pequeño repositorio donde aprenderemos un par de trucos para trabajar rápidamente con Visual Studio Code.

## Topicos
* Edicion basica de codigo
* Multicursores
* Definiciones de clases
* Snippets
* Extensiones
